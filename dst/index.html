class Quest

Единственный верный вариант ответа на каждый вопрос, по результатам выставляется оценка и выносится вердикт

class QuestGuess

У каждого вердикта есть набор ответов на вопросы, выбирается вердикт с наиболее совпадающими ответами

class QuestMdz

Похож на Quest, но после каждого ответа показывается правильный вариант и статистика ответов. Это ближе всего к тому, что сделано на Медузе
